# Stay
Stay是一个开源的iOS Safari浏览器插件，做为一个浏览器插件示例的同时也已经上线AppStore，并且保证免费。

Stay需要运行在iOS 15及以上。


## 主要功能
- 广告拦截
- 阻止应用跳转
- 自动展开文本


## 贡献脚本
#### 苹果开发者
你可以下载源代码并且改造相关的功能部署在你的手机设备上。

#### 非苹果开发者
如果你有前端开发经验，你也可以选择贡献脚本，当PR被合并后，你也会在更新的Stay应用中体验到你更新的功能。
关于如何贡献脚本可以点击这里。
